## 🌌 About the Project

**Which Planet Are You?** is an interactive web-based personality quiz that presents users with a series of questions and determines their matching planet based on their responses.

Built with a stunning **dark cosmic theme** featuring animated stars and glowing effects, this project delivers both a fun experience and a visually immersive UI.

🔗 **Live Demo:** [waniaadnan.github.io/Personality_Quiz-](https://waniaadnan.github.io/Personality_Quiz-/)

---

## ✨ Features

- 🔭 Dynamic question handling with smooth transitions
- 🧮 Score calculation and planet-matching logic
- 🌠 Animated star background with glowing effects
- 📱 Responsive and visually appealing design
- ⚡ Pure front-end — no libraries or frameworks needed

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure & layout |
| CSS3 | Styling, animations, cosmic theme |
| JavaScript | Quiz logic, DOM manipulation, event handling |

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/WaniaAdnan/Personality_Quiz-.git
   ```
2. Open `index.html` in your browser — that's it!
