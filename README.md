## 🌌 About the Project

```python
class PersonalityQuiz:
    def __init__(self):
        self.name        = "Which Planet Are You?"
        self.type        = "Interactive Web Quiz"
        self.theme       = "Dark Cosmic — animated stars & glowing effects"
        self.features    = ["Dynamic questions", "Score-based matching", "Cosmic UI", "Responsive design"]
        self.built_with  = ["HTML5", "CSS3", "JavaScript"]
        self.dependency  = None  # Pure front-end — zero libraries needed
        self.live_at     = "waniaadnan.github.io/Personality_Quiz-"

    def describe(self):
        print("Answer the questions. Discover your planet. 🪐")

quiz = PersonalityQuiz()
```
